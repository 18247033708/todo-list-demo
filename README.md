# todo-demo
本项目使用vue3 + ts + ant-design-vue 实现 Todo 功能
目前为第一版


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
